# 🏥Hospital-Management-System-SQL-Project
An end-to-end SQL project that analyzes a Hospital Management System database using MySQL. This project focuses on solving real-world business problems related to patients, doctors, appointments, rooms, beds, revenue, and staff performance.

---

## 📌 Project Overview

This project demonstrates how SQL can be used to analyze hospital data and generate meaningful business insights. It includes beginner to advanced SQL queries covering data retrieval, reporting, and analytics.

---

## 🎯 Objectives

- Analyze patient and appointment data
- Monitor doctor performance
- Track room and bed utilization
- Calculate hospital revenue
- Evaluate staff working hours
- Solve real-world business problems using SQL
